# FDE管理平台

面向研发团队的现代化任务管理平台，以 **流动看板 + AI 协作者** 实现持续价值交付。

## 核心特性

| 模块 | 能力 |
|------|------|
| **管理看板** | 领导全局概览，四态统计，快捷跳转 |
| **流动看板** | 列表视图 / 四列看板切换，状态筛选，人员筛选，CSV 导出 |
| **四态流转** | 执行中 · 阻塞 · 已归档 · 已终止（终止/二次分配/退回执行中） |
| **审核备案** | 需求审核与人员分配 |
| **归档查看** | 已完成任务归档与退回 |
| **需求提交** | 文本/文档/语音/会议录音智能拆解 |
| **AI 助手** | 需求拆分、站会摘要、风险扫描、智能对话 |
| **个人主页** | 成员档案与参与任务 |

## 快速启动

## 环境配置（可选）

复制 `.env.example` 为 `.env`：

```bash
cp .env.example .env
```

| 变量 | 用途 |
|------|------|
| `DEEPSEEK_API_KEY` | 大模型分析、拆任务、对话 |
| `DASHSCOPE_API_KEY` | 语音转写（Paraformer） |

未配置时自动使用本地规则引擎。

## 技术栈

- **后端**: Node.js + Express + JSON 持久化
- **前端**: 原生 HTML/CSS/JS SPA
- **AI**: DeepSeek / 通义千问 + 阿里云 ASR（可选）
