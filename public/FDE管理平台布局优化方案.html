1. 背景与目标

核心判断：FDE管理平台在任务四态流转、审核分配、AI 协作者、列表导出等方面功能已完备。当前主要痛点不在「缺能力」，而在入口过多、页面重叠、角色边界不清，导致不同职能人员难以快速找到「今天最该用的页面」。

优化目标

  • 领导/审核人：登录即进入可操作的「任务中心」，一屏处理执行中 / 阻塞 / 归档 / 终止任务
  • 执行人员：聚焦「今日工作台」，每日只需提交进展、标记完成
  • 所有人：减少重复按钮与重复列表，保留全部后端能力与 API

2. 现状诊断

2.1 导航过多

角色优化前侧边栏主要问题
领导/审核人9 项管理看板、流动看板、审核备案、归档查看信息高度重复
执行人员4 项流动看板与「我的工作台」边界模糊
多职能用户三类能力叠加不知默认从哪个页面开始一天的工作

2.2 功能页面重叠

优化前页面关系（简化） 管理看板 ──→ 流动看板 ──→ 审核备案 │ │ └──────→ 归档查看 个人主页任务表 ≈ 流动看板列表 ≈ 审核备案卡片 顶部「站会/风险/新建」在管理看板、流动看板重复出现 侧边栏四态徽章 ≈ 管理看板统计 ≈ 流动看板筛选按钮

2.3 操作路径偏长

领导处理一个阻塞任务，优化前典型路径：

管理看板→ 流动看板→ 筛选阻塞→ 操作

执行人员每日核心动作仅两项：提交进展、标记完成，却分散在「我的工作台」与「流动看板」。

3. 优化原则

① 按角色给默认首页

登录即进入该角色「今天最该用的页面」，而非统一仪表盘。

② 一任务一入口

同类任务列表只保留一处主战场，避免三处表格维护同一数据视图。

③ 高频前置、低频折叠

每日操作用大按钮；AI、团队、归档详情收进二级 Tab 或折叠区。

④ 看与做分离

「查看」用只读详情 + 每日进展时间线；「编辑/审核」保留独立入口。

⑤ 不删能力，只合并入口

后端 API、四态流转、权限模型、导出与 AI 引擎全部保留，仅调整前端路由与布局。

5. 核心页面设计

5.1 任务中心（领导主战场）

┌─────────────────────────────────────────────────────────┐ │ 任务中心 [站会摘要] [风险扫描] [+ 新建] │ ← 仅顶部栏 ├─────────────────────────────────────────────────────────┤ │ ⚠ 有 N 项阻塞任务待处理 [立即查看] │ ← 阻塞告警条 ├─────────────────────────────────────────────────────────┤ │ [全部] [执行中] [阻塞] [已归档] [已终止] 审核人▼ 执行人▼ │ │ [列表视图] [四列看板] [📥 导出清单] │ ├─────────────────────────────────────────────────────────┤ │ 表格 / 看板（复用原 renderKanban 组件） │ │ 列：编号 · 名称 · 状态 · 人员 · 起止时间 · 可执行操作 │ └─────────────────────────────────────────────────────────┘
  • 四态 Tab 即原流动看板筛选，无需再进「审核备案」
  • 列表「可执行操作」列保留：终止 / 二次分配 / 退回执行中 / 任务完成
  • 有阻塞时侧边栏四态徽章高亮红色提醒

5.2 今日工作台（执行人员主战场）

┌─────────────────────────────────────────────────────────┐ │ 今日工作台 [+ 上报进展] │ ├─────────────────────────────────────────────────────────┤ │ 摘要条：待办 3 · 协助 1 · 今日未报进展 2 · [提交需求] │ ├─────────────────────────────────────────────────────────┤ │ ▼ 任务卡片（精简） │ │ 标题 / 状态 / 最近进展一行 │ │ [今日进展输入框] │ │ [提交] [完成] [查看进展] │ └─────────────────────────────────────────────────────────┘

5.3 需求与 AI（领导）

📤 新建需求 🤖 AI 工具

「新建需求」复用原需求提交表单 + AI 助手；「AI 工具」复用原 AI 助手页（拆分、站会、风险、对话、洞察历史）。

5.4 提交需求（执行人员）

📝 快速提交 🎙️ 会议/语音 📋 提交记录

默认展开「快速提交」表单；会议语音与 AI 拆解收入第二个 Tab，降低首屏复杂度。

5.5 我的 / 我的记录

  • 顶部摘要条：状态、进行中、协助、审核（领导）、阻塞数，一键跳转任务中心或工作台
  • 两个任务表（参与任务、参与审核）默认各显示 5 条，可「展开全部」
  • 「查看」打开只读详情，含每日进展提交记录时间线；支持单条下载
  • 移除与任务中心重复的「手上任务 / 当前卡点」双栏卡片
